102下 JAVA期末專題   
===========   
   
題目說明:
-------------  
一個簡單的轉速時速轉換器
讓使用者輸入轉速或者時速，接著選擇檔位
即可完成換算

其分析的車款為Kawasaki Ninja 250
齒比為網路上找來的，有錯不要怪我XD

   
程式碼說明:   
-------------   
不囉嗦，看圖
![UML][1]

編譯&執行:
------------- 
開啟CMD然後cd到目錄後輸入
```
javac -encoding UTF-8 -d . *.java
```
完成編譯

接著在原目錄執行
```
java finalProject/main
```
啟動程式
